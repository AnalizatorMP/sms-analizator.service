1 🔐 Регистрация в системе

Шаг 1: Найдите наш Telegram бот
  • Перейдите по ссылке: @SMS_Analizator_bot
  • Нажмите кнопку "Запустить" или отправьте команду /start
Шаг 2: Отправьте контакт
  • Нажмите на кнопку "Отправить контакт" в боте
  • Подтвердите отправку вашего номера телефона
  • Дождитесь сообщения с логином и паролем
Шаг 3: Вход в систему
  • Откройте веб-интерфейс системы
  • Введите полученный логин (номер телефона) и пароль
  • Нажмите "Войти"
💡 Важно: Сохраните логин и пароль в надежном месте. Логином является ваш номер телефона без символа "+".

2 📱 Настройка Telegram каналов

Шаг 1: Создайте Telegram канал
  • Откройте Telegram
  • Создайте новый канал: Меню → Новый канал
  • Дайте каналу понятное название (например, "SMS Уведомления")
  • Сделайте канал приватным
Шаг 2: Добавьте бот в канал
  • Перейдите в настройки канала → Администраторы
  • Нажмите "Добавить администратора"
  • Найдите @SMS_Analizator_bot
  • Добавьте бот и дайте ему права на отправку сообщений
Шаг 3: Зарегистрируйте канал в системе
  • Напишите боту в личные сообщения
  • Отправьте команду для добавления канала
  • Следуйте инструкциям бота
⚠️ Внимание: Убедитесь, что бот имеет права администратора в канале, иначе он не сможет отправлять сообщения.

3 ⚙️ Настройка SMS сервисов

Шаг 1: Перейдите в "Настройка сервисов"
  • В левом меню нажмите "Настройка сервисов"
  • Выберите ваш SMS провайдер из списка
Шаг 2: Для Novofon
⚠️ Важно: Для NovoFon необходимо настроить уведомления в личном кабинете провайдера. Следуйте подробной инструкции ниже.
2.1. Настройка в системе SMS Анализатор:
  • Выберите "Novofon" в выпадающем списке
  • Оставьте поля "Название" и "Ключ" пустыми
  • В поле "Телефон" введите номер без "+" (например: 79261234567)
  • Нажмите "Отправить"
  • Скопируйте webhook URL, который появится в разделе "Ваши сохраненные сервисы"
2.2. Настройка уведомлений в личном кабинете NovoFon:
Шаг 1Зайдите в настройки
  • Войдите в личный кабинет NovoFon
  • В левом меню найдите раздел "Настройки"
  • Нажмите на "Настройки"
Скриншот 1: Меню настроек
Шаг 2Перейдите в "Настройки уведомлений"
  • В разделе настроек найдите "Уведомления"
  • Нажмите на "Уведомления"
  • Вы увидите список текущих уведомлений
Скриншот 2: Раздел уведомлений
Шаг 3Добавьте новое уведомление
  • Нажмите кнопку "Добавить уведомление"
  • Откроется форма настройки уведомления
Скриншот 3: Добавить уведомление
Шаг 4Настройте параметры уведомления
  • Название уведомления: введите любое удобное название (например: "SMS Анализатор")
  • Тип события: обязательно выберите "Получено SMS сообщение"
  • Активно: убедитесь, что переключатель включен
Скриншот 4: Настройки уведомления
Шаг 5Настройте способы уведомления
📡 HTTP уведомление:
  • Включите HTTP: активируйте переключатель для HTTP уведомлений
  • Метод: выберите POST
  • URL: вставьте webhook URL, скопированный из раздела "Ваши сохраненные сервисы"
Шаг 6Настройте тело запроса
📝 Тело запроса: В поле "Тело" введите точно следующий JSON:
{% verbatim %}{"result":{"caller_did":{{virtual_phone_number}},"caller_id":{{contact_phone_number}},"text":{{message}}}}{% endverbatim %}
⚠️ Важно: скопируйте этот JSON точно, включая все скобки и кавычки
Шаг 7Сохраните настройки
  • Нажмите кнопку "Сохранить" или "Добавить параметр"
  • Убедитесь, что уведомление появилось в списке и помечено как активное
  • Настройка NovoFon завершена!
✅ Готово! Теперь при получении SMS на ваш номер NovoFon, уведомление будет автоматически отправляться в нашу систему для дальнейшей пересылки в Telegram.
Шаг 3: Для Telfin и Mango
  • Выберите соответствующий сервис
  • Введите название (любое удобное для вас)
  • Введите API ключ, полученный в личном кабинете провайдера
  • Нажмите "Отправить"
✅ Результат: Добавленные сервисы появятся в разделе "Ваши сохраненные сервисы" с кнопкой для копирования webhook URL.

4 📋 Создание правил пересылки

Шаг 1: Перейдите в "Настройка правил"
  • В левом меню нажмите "Настройка правил"
  • Здесь вы создадите правила для автоматической пересылки
Шаг 2: Настройте отправителя
  • В поле "Отправитель" введите имя отправителя SMS (например: "SBERBANK", "VK", "GOOGLE")
  • Или поставьте галочку "Любой отправитель" для пересылки всех SMS
Шаг 3: Выберите номер
  • В выпадающем списке "Выберите телефон" выберите номер, на который приходят SMS
  • Этот номер должен быть предварительно добавлен в настройках сервисов
Шаг 4: Выберите канал
  • В списке "Выберите канал Telegram" выберите канал для пересылки
  • Канал должен быть предварительно настроен через Telegram бот
Шаг 5: Сохраните правило
  • Нажмите кнопку "Отправить"
  • Правило появится в разделе "Ваши сохраненные правила"
💡 Совет: Создавайте отдельные правила для разных типов SMS. Например, одно правило для банковских SMS, другое для соцсетей.

5 🔗 Настройка webhook в SMS провайдере

Шаг 1: Скопируйте webhook URL
  • Перейдите в "Настройка сервисов"
  • Найдите ваш добавленный сервис
  • Нажмите кнопку "Скопировать webhook URL"
  • URL будет скопирован в буфер обмена
Шаг 2: Настройка в Novofon
  • Войдите в личный кабинет Novofon
  • Перейдите в раздел "Настройки" → "Webhook"
  • Вставьте скопированный URL
  • Выберите события: "Входящие SMS"
  • Сохраните настройки
Шаг 3: Настройка в Telfin
  • Войдите в панель управления Telfin
  • Найдите раздел "Уведомления" или "Webhook"
  • Добавьте новый webhook с скопированным URL
  • Укажите тип события: "SMS получен"
Шаг 4: Настройка в Mango
  • Зайдите в настройки Mango Office
  • Перейдите в "Интеграции" → "Webhook"
  • Создайте новый webhook с вашим URL
  • Выберите события SMS уведомлений
🔒 Безопасность: Webhook URL содержит уникальный токен. Не передавайте его третьим лицам!

6 🧪 Тестирование системы

Шаг 1: Отправьте тестовое SMS
  • Отправьте SMS на настроенный номер
  • Или попросите кого-то отправить вам SMS
  • Убедитесь, что отправитель соответствует вашим правилам
Шаг 2: Проверьте Telegram канал
  • Откройте настроенный Telegram канал
  • SMS должно появиться в течение 1-2 минут
  • Сообщение будет содержать текст SMS и информацию об отправителе
Шаг 3: Проверьте логи
  • При возникновении проблем обратитесь к администратору
  • Логи помогут определить причину неработающей пересылки
🎉 Поздравляем! Если SMS успешно пересылается в Telegram, система настроена правильно.

7 🔧 Решение типичных проблем

  • Проверьте правильность webhook URL в настройках провайдера
  • Убедитесь, что бот добавлен в канал как администратор
  • Проверьте соответствие отправителя SMS вашим правилам
  • Убедитесь, что правило создано для правильного номера

  • Убедитесь, что вводите номер телефона без символа "+"
  • Проверьте правильность пароля (пароли чувствительны к регистру)
  • Попробуйте заново зарегистрироваться через Telegram бот

  • Убедитесь, что вы пишете правильному боту: @SMS_Analizator_bot
  • Попробуйте отправить команду /start
  • Проверьте, не заблокировали ли вы бота случайно
  • Если проблема сохраняется, обратитесь в поддержку

8 🚀 Расширенные функции

Множественные правила
  • Создавайте несколько правил для одного номера
  • Настраивайте разные каналы для разных отправителей
  • Используйте правило "Любой отправитель" как резервное
Управление правилами
  • Удаляйте неактуальные правила кнопкой "Удалить"
  • Регулярно проверяйте список активных правил
  • Правила работают в порядке создания
Безопасность
  • Регулярно меняйте пароль
  • Не передавайте webhook URL третьим лицам
  • Используйте надежные пароли
💡 Совет: Ведите записи о ваших настройках и правилах для быстрого восстановления в случае необходимости.
